## Description
A Java-based Temperature Converter that allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin. The project uses standard conversion formulas and provides a simple console-based interface.

## Features
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius
- User-friendly interface
- Accurate calculations

## Technologies Used
- Java
- Scanner Class
- Conditional Statements
- Arithmetic Operations
